第一小结：简单的Git命令

简单的git命令是程序员的必不可少的知识。

首先就是安装gitbash

打开命令行当前显示的 ~ 表示用户主目录。

git基本命令如下：

  • cd 就是直接跳转到用户主目录。

  • ls 查看当前目录下的内容。

  • ls -s 查看对应文件的命令权限

  • cd ../ 或 cd .. 回到上一级

  • cd/ 回到顶级目录

  • cd D: 跳到D盘

  • cd .test 跳到test文件

  • mkdir test text 当前目录下创建文件夹test 和 text

  • mkdir test/text 在文件夹test下创建text文件夹，test文件夹不存在的时候需要加参数 p 如下：

  • mkdir -p test/text

  • touch index.html 创建一个index.html文件

  • rm index.html 删除文件

  • rm test -r 删除文件夹

  • rm test -rf 强制删除文件夹

  • rm test/* -r 删除test文件夹下的所有文件夹

  • cp text test -r 把text复制到test文件夹下

  • mv index.html text 把index.html剪切到text文件夹中

  • mv text test 将text文件夹命名为test

  • cat index.html 预览index.html文件

  • du -sh index.html 查看test文件夹的大小

  • vi index.html 进入index.html文件

  • git clone git@github.com:xiaoyu311/xiaoyu311.github.io.git mygit 克隆远端仓库到本地mygit文件夹下

  • git log 查看版本历史

  • git status 查看仓库状态

  • git add . 跟踪所有修改

  • git commit -m 'my commit' 版本留言

  • git push 推送到远端仓库

在推送远端仓库时候，一般会git pull 来确保本地仓库已经是最新版本。

  • git diff 查看修改

  • git checkout 还原修改

  • git checkout . 还原所有修改

  • git init 初始化一个仓库，这时候会生成一个.git 隐藏文件夹。

  • ssh-keygen 将会生成密钥文件和私钥文件，id_rsa,id_rsa.pub或id_dsa,id_dsa.pub。

这时候需要把生成的公钥，也就是id_dsa.pub文件中的字符拷贝到远端git仓库中，步骤如下：

  • settings=>SSH and GPG keys => New SSH key
